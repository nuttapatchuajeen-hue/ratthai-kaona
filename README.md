# รัฐไทยก้าวหน้า — เทคโนโลยีภาครัฐไทย

โครงงานเว็บไซต์ที่รวบรวมข้อมูลภาครัฐไทยให้เข้าใจง่ายในที่เดียว — **กรม · กระทรวง · คณะรัฐมนตรี · ผู้แทนราษฎร · สถิติ**

จัดทำโดย **ณัฐภัทร ช่วยจีน** (Master Nook)

## โครงสร้างเว็บ
| เส้นทาง | เนื้อหา |
|--------|---------|
| `/` → `/hub/` | หน้าหลัก + เกี่ยวกับเรา + วิดีโอ |
| `/structure.html` | แผนผังโครงสร้างรัฐไทย (กระทรวง/กรม/หน่วยงาน) |
| `/election/` | ส.ส. ผู้แทนราษฎร — รายเขต / บัญชีรายชื่อ / คะแนนพรรค |
| `/cabinet/` | ทำเนียบคณะรัฐมนตรี |
| `/stats/` | แดชบอร์ดสถิติข้อมูลรัฐ |

## เทคโนโลยี
HTML / CSS / JavaScript (vanilla) · GSAP · D3.js · Lenis
